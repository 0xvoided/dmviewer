# DM Viewer System

Click [Here](https://github.com/0xvoided/SyntaxCloner/archive/refs/heads/main.zip) to download

## Professional Chat Visualization

DM Viewer is a standalone web-based interface designed to transform your raw text logs into a clean, readable, and professional chat experience. It processes your data locally on your machine for maximum privacy.

* 🎨 Modern UI: Features a dark, high-tech gothic aesthetic.
* 🛡️ Privacy First: All processing happens in your browser; no data is ever uploaded.
* 📊 Smart Analytics: Automatically calculates message counts and sent/received ratios.
* 📅 Date Parsing: Full support for both English and Arabic timestamp formats.
* 🔍 Live Search: Quickly find specific channels or users within your logs.

## Built With

* HTML5
* CSS3 (Custom Shell UI)
* Pure JavaScript (No external APIs)

## Getting Started

You don't need to install anything. Just have your chat logs ready.

## Setup

1. Download the Viewer file:
viewer.html


2. Open the file in any modern web browser (Chrome, Brave, Edge).

3. Locate your log files (usually found in the /logs folder of SyntaxCloner).

## 🚀 How to Use

To visualize your data in the interface:

1. Drag your `.txt` or `.json` log file and drop it anywhere on the Viewer page.
2. The system will automatically parse the names, dates, and messages.
3. Select a channel from the sidebar to start reading.
